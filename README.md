# Expense-Tracker-CLI
The Expense Tracker CLI is a simple command-line based application built using Python that helps users manage their daily income and expenses efficiently. This project allows users to record financial transactions, store them in a CSV file, and generate a quick summary of their financial status.

📊 Expense Tracker CLI (Python)
📌 Description
The Expense Tracker CLI is a simple command-line based application built using Python that helps users manage their daily income and expenses efficiently. This project allows users to record financial transactions, store them in a CSV file, and generate a quick summary of their financial status.
It is designed as a beginner-friendly project to demonstrate core Python concepts such as file handling, data processing, and user interaction through the command line.

🚀 Features
➕ Add income and expense entries
📅 Auto date entry (or manual input)
📂 Stores data in a CSV file
📊 View total income, expenses, and balance
🧾 Simple and easy-to-use CLI interface

🛠️ Technologies Used
Python 🐍
CSV (File Handling)
Datetime Module

📁 Project Structure
Expense-Tracker/
│── expense_tracker.py
│── expenses.csv (auto-created)
│── README.md

Navigate to the folder:
cd Expense-Tracker
Run the script:
python expense_tracker.py
